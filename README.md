## ğŸ“– How It Works

### ReAct Loop (Thought â†’ Action â†’ Observation â†’ Repeat)

1. **Thought**: Agent analyzes the user request and plans steps
2. **Action**: Agent executes a tool or generates content
3. **Observation**: Agent observes the result
4. **Repeat**: Continue until task is complete

### Artifacts

Artifacts are concrete deliverables generated by the agent:

- Code files
- Documents (Markdown, HTML)
- Charts and visualizations
- Data tables
- Interactive components
